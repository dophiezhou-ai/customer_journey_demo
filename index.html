What is Agent OS

官网负责让客户理解价值，用户中心负责让客户连接数据、完成认证、试用 Agent，并在看到结果后升级订阅。

Six Agent Roles

6 大 Agent 各自解决一个经营场景

不是所有客户一开始都需要 6 个 Agent。Dowsure 会基于店铺规模、增长阶段和经营痛点推荐合适组合。

Insight Agent

增长机会

识别高潜 SKU、类目趋势、关键词机会和可放量方向。

适合：想找新增量的成长卖家
Spend Agent

支出效率

分析广告、物流、平台费和低效花费，发现利润被吃掉的位置。

适合：广告和费用上升的卖家
Deal Agent

采购与账期

分析供应商账期、采购订单、补货节奏和 SKU 利润。

适合：供应商付款压力明显的卖家
Capital Agent

资金节奏

判断回款、库存、广告和备货之间的现金流错配。

适合：回款慢、备货急、旺季要放量的卖家
Treasury Agent

账户与币种

分析多币种、换汇、汇损和账户余额调度效率。

适合：多平台、多币种、多账户卖家
Sentinel Agent

风险守护

监控店铺健康、授权异常、资料完整度和准入边界。

适合：所有需要稳健增长的卖家
Embedded Trio

TRIO 是 Agent 诊断后的场景化服务

Dowsure 的核心服务是 AI Agent OS。FastPay、TermPay、MoreFund 会在客户经营数据触发对应场景后，再作为服务内的后续评估入口出现。

FastPay

店铺应收账款回流慢

当 Amazon 回款尚未到账，但近期要补货、投广告或支付账单时，由 Capital Agent 判断是否适合进入 FastPay 评估。

TermPay

下游账期过长

当采购付款、供应商账期和订单回款之间存在明显时间差时，由 Deal + Capital Agent 建议账期安排。

MoreFund

旺季备货或爆品加单

当 Insight 发现增长窗口、Spend 发现投放效率、Capital 发现备货缺口后，再判断是否需要 MoreFund 支持。

Customer Proof

客户先看到和自己相似的场景，再决定是否体验

官网中的案例用于建立信任，用户中心中的 ROI Proof 用于推动试用转付费、付费转升级。
