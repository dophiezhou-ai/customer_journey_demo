从经营数据到 Agent 行动建议

系统不会要求卖家一次购买所有 Agent，而是根据店铺阶段、数据完整度和经营问题推荐启用顺序。

Insight Agent

增长机会

识别高潜 SKU、类目趋势、平台增长机会和可放量方向。

Spend Agent

支出效率

分析广告、物流、平台费和低效花费，给出预算调整动作。

Deal Agent

采购与利润

分析供应商账期、采购订单、SKU 利润和补货节奏。

Capital Agent

资金节奏

判断回款、库存、广告和备货之间的现金流错配。

Treasury Agent

币种与账户

分析多币种、换汇、汇损和账户资金效率，适合成熟卖家升级。

Sentinel Agent

风险守护

监控店铺健康、退货、授权异常、资料完整度和准入边界。
